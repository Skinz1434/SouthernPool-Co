# Deployment Instructions for SouthernPool Co

## Step 1: Create GitHub Repository
1. Go to https://github.com/new
2. Repository name: southernpool-co
3. Description: Premium pool construction company website - Louisiana's finest pool builders
4. Make it PUBLIC
5. DO NOT initialize with README, .gitignore, or license
6. Click "Create repository"

## Step 2: Push to GitHub
After creating the repository, run these commands in Git Bash or terminal:

```bash
cd F:/SouthernPoolCo

# Add remote origin (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/southernpool-co.git

# Push to GitHub
git branch -M main
git push -u origin main
```

## Step 3: Deploy to Vercel

### Option A: Via Vercel CLI
```bash
# Install Vercel CLI if not installed
npm i -g vercel

# Deploy
cd F:/SouthernPoolCo
vercel

# Follow prompts:
# - Link to existing project? No
# - What's your project name? southernpool-co
# - In which directory is your code? ./ (current directory)
# - Want to override settings? No
```

### Option B: Via Vercel Website
1. Go to https://vercel.com/new
2. Import Git Repository
3. Select your GitHub account
4. Import southernpool-co repository
5. Configure Project:
   - Framework Preset: Other
   - Root Directory: ./
   - Build Command: (leave empty)
   - Output Directory: (leave empty)
6. Click "Deploy"

## Step 4: Custom Domain (Optional)
Once deployed, you can add a custom domain:
1. Go to Vercel Dashboard
2. Select your project
3. Go to Settings > Domains
4. Add your custom domain

## URLs After Deployment
- GitHub: https://github.com/YOUR_USERNAME/southernpool-co
- Vercel: https://southernpool-co.vercel.app

## Notes
- The site is static HTML/CSS/JS, so no build process is needed
- Images are included in the repository
- For Lottie animations to work, you'll need to add the JSON files to /animations folder
